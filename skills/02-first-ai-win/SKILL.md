# Your First AI Win

Guide the user through completing one real work task with AI. Offer three walkthroughs and let them pick one.

## Walkthrough A: Batch-Draft Email Replies

Ask the user to paste 2-3 emails with a one-sentence intent for each. Then draft all replies:

- Keep each reply under 150 words unless the situation demands more
- Match the formality level of the original sender
- Use the user's custom instructions for voice and style

## Walkthrough B: Summarise a Meeting

Ask the user to paste meeting notes or a transcript. Produce:

1. **KEY DECISIONS** - each as a single clear sentence
2. **ACTION ITEMS** - task, owner, deadline (TBD if not stated)
3. **OPEN QUESTIONS** - raised but not resolved
4. **SUMMARY** - 3-5 sentences for someone who was not there

## Walkthrough C: Turn Bullets into a Memo

Ask the user for: rough notes, document type, audience, target length. Then:

1. Organise the information logically
2. Write in the appropriate style for the audience
3. Highlight gaps where more information is needed
4. Mark uncertain content as [NEEDS INPUT]

## Refinement Pattern

When any draft needs adjustment, steer with follow-ups instead of rewriting manually. Ask the user what to change and apply targeted adjustments. This is the core habit the book builds.
