# Frequently Asked Questions — AI For Everyday Automation

Common questions about the **AI automation skills** that accompany the book *[AI for Everyday Automation](https://www.amazon.com/AI-Everyday-Automation-Workflows-Hours/dp/1808089812/)* by [Dipankar Sarkar](https://www.dipankar.name).

## What is AI For Everyday Automation?

AI For Everyday Automation is a free collection of 10 ready-to-use AI skills that automate everyday knowledge work — email, meetings, research, document drafting, task tracking, data analysis, and content creation. Each skill is a plain-text instruction file you upload to Claude, ChatGPT, or Perplexity. It is the official companion to the Packt book of the same name.

## What is an AI skill?

An AI skill is an instruction set that teaches your AI tool how to perform a specific task. You upload it once, and from then on the AI knows how to handle that work whenever you ask in plain language — no prompts to memorise and nothing to copy-paste.

## Do I need to install all 10 skills?

No. Start with the one that saves you the most time — usually Inbox Zero (03) or Meetings (04) — and add more as you get comfortable.

## Can I use these with tools other than Claude?

Yes. The skill ZIPs work with Claude, ChatGPT, and Perplexity. The workflows are the same across all three. Some features (such as Google Drive integration) behave differently depending on the tool — see the book for tool-specific notes.

## Will the AI send emails without my permission?

No. Every skill that sends an email or creates a task shows you a preview first and waits for your explicit approval. You are always the final gate.

## What if the AI gets something wrong?

Every skill includes review steps, so you check the output before it goes anywhere. The book covers common failure modes — wrong tone, invented facts, misattributed tasks — and how to fix them with follow-up instructions.

## Do I need to pay for AI tools?

No. You can follow every workflow on free tiers. Paid tiers (Claude Pro or ChatGPT Plus) remove friction with higher limits, file uploads, and more integrations. One paid subscription is usually enough.

## Do I need any technical or coding knowledge?

No. If you can download a file and click "upload," you can install and use these skills. The `SKILL.md` files are plain text you can read and edit in any text editor — no code.

## What if my workplace blocks AI integrations?

Every skill also runs in manual mode: instead of the AI reading your inbox directly, you paste the email into the conversation. The instructions are identical; connected mode just saves the copy-paste step.

## How do I update a skill?

Download the new ZIP and upload it again — the old version is replaced. Check this repository for updated skills as AI tools evolve.
