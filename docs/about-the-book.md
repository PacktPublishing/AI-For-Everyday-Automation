## What the book is about

*AI for Everyday Automation* is a practical, non-technical guide to using AI tools like Claude, ChatGPT, and Perplexity to automate the repetitive knowledge work that fills a normal work week — email, meetings, research, writing, task tracking, data, and content. Instead of theory, it walks you through hands-on workflows you can run the same day, and shows how individual time savings compound into a full weekly system.

Readers report saving **8–12 hours in their first full week** once the system is running.

## Who it is for

- Professionals, managers, and operators who want to automate routine work without learning to code
- Teams standardising how they use AI for email, meetings, and reporting
- Anyone new to AI tools who wants a guided, real-world starting point

## What you'll learn

- How to **audit your week** and identify the highest-leverage tasks to automate
- How to run **AI inbox triage** and draft replies in your own voice
- How to turn **meeting transcripts** into summaries, action items, and follow-ups
- How to produce **sourced research briefs** and **first-draft documents** in minutes
- How to **extract and track tasks** automatically across email, meetings, and docs
- How to **analyse spreadsheets** and **generate content** from a single update
- How to assemble it all into an **automated Monday-to-Friday workflow**

This repository contains every workflow from the book, packaged as ready-to-install skills.
