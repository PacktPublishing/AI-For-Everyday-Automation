# Data Analysis — Clean, Chart, and Summarise Spreadsheets With AI

**Book chapter:** 8 · **Download:** [`08-data-analysis.zip`](../../dist/08-data-analysis.zip) · **Source:** [`SKILL.md`](../../skills/08-data-analysis/SKILL.md)

## What this skill does

Upload a spreadsheet, CSV, or Google Sheet and this skill **cleans the data, runs the analysis, builds charts, and explains the results in plain English**. Use it for budget-versus-actuals, variance, revenue breakdowns, survey summaries, and pivot tables — no formulas required.

## Say something like

- *"Analyse this spreadsheet"*
- *"Clean this data"*
- *"Chart this"*
- *"What does this data show?"*
- *"Budget variance"*
- *"Read this Google Sheet"*

## Integrations required

**Code execution**, and optionally **Google Drive** to read sheets directly.

## Part of a system

This is skill 08 of 10 in **AI For Everyday Automation**. It analyses task data from [Task Extraction (07)](07-task-extraction.md) and feeds charts into [First Drafts (06)](06-first-drafts.md) and [Content on Demand (09)](09-content-on-demand.md).
