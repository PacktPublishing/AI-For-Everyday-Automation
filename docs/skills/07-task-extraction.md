# Task Extraction — Pull Action Items From Email, Meetings, and Docs

**Book chapter:** 7 · **Download:** [`07-task-extraction.zip`](../../dist/07-task-extraction.zip) · **Source:** [`SKILL.md`](../../skills/07-task-extraction/SKILL.md)

## What this skill does

This skill finds the **hidden tasks and commitments** buried in your meetings, emails, documents, and messages, then tracks them in your task tool. Run a weekly task sweep, reconcile your task portfolio, and catch untracked work before it slips.

## Say something like

- *"Extract tasks from this"*
- *"What did I commit to this week?"*
- *"Run my task sweep"*
- *"Check for untracked tasks"*
- *"Reconcile my tasks"*

## Integrations required

**Gmail, Google Drive, and a task tool** (Asana / Linear / Notion). Manual mode also supported.

This is skill 07 of 10 in **AI For Everyday Automation**. It draws on [Inbox Zero (03)](03-inbox-zero.md) and [Meetings (04)](04-meetings.md), and its task data feeds [First Drafts (06)](06-first-drafts.md) and [Data Analysis (08)](08-data-analysis.md).
