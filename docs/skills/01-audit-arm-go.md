# Audit, Arm, Go — Find What to Automate With AI

**Book chapter:** 1 · **Download:** [`01-audit-arm-go.zip`](../../dist/01-audit-arm-go.zip) · **Source:** [`SKILL.md`](../../skills/01-audit-arm-go/SKILL.md)

## What this skill does

Audit, Arm, Go is the starting-point skill. It helps you **audit your work week**, identify the tasks worth automating with AI, choose the right AI tool for each task, and set up your AI workbench with custom instructions. Run this first if you are not sure where to begin.

## Say something like

- *"Audit my week"*
- *"What should I automate?"*
- *"Which AI tool should I use for this?"*
- *"Help me set up my custom instructions"*
- *"Help me get started with AI"*

## Integrations required

None — this skill works in any AI tool out of the box.
