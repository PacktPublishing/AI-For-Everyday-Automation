# Documentation — AI For Everyday Automation

Public documentation for the **AI For Everyday Automation** skill collection — the official companion to the book *[AI for Everyday Automation: 7 AI Workflows to Save Hours at Work Every Week](https://www.amazon.com/AI-Everyday-Automation-Workflows-Hours/dp/1808089812/)* by [Dipankar Sarkar](https://www.dipankar.name) (Packt Publishing).

These ready-to-use AI skills work in **Claude**, **ChatGPT**, and **Perplexity**. Install one in under two minutes and automate email, meetings, research, reports, tasks, data, and content.
